## 🧭 Making the Switch: Cloud PMS Migration Checklist

Ready to move from legacy to cloud? Use this checklist:

### ✅ Pre-Migration Planning:

- [ ] Audit current system data (guests, bookings, rates, reports)
- [ ] Identify must-have features for your property
- [ ] Set budget and timeline for migration
- [ ] Assign internal migration champion (front desk manager or owner)
- [ ] Schedule migration during low-occupancy period

### ✅ During Migration:

- [ ] Export all data from legacy system
- [ ] Import data into cloud PMS (provider handles this)
- [ ] Verify data accuracy (sample check bookings, guest profiles, rates)
- [ ] Configure settings (room types, rates, user permissions, integrations)
- [ ] Test all workflows (check-in, check-out, billing, reports)

### ✅ Post-Migration:

- [ ] Train all staff on new system
- [ ] Run parallel system for 2-3 days (optional safety net)
- [ ] Go live with cloud PMS fully
- [ ] Monitor for any issues in first week
- [ ] Schedule follow-up training after 30 days
